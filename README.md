# 🎓 Student Management System

> A clean, browser-based web application to manage student records — built with pure HTML, CSS & JavaScript.

---

## 📌 Overview

The **Student Management System** is a front-end web project designed to simplify the process of tracking student data. From adding new students and marking attendance to calculating grades — everything runs directly in the browser with zero backend required.

All data is persisted using **LocalStorage**, so records survive page refreshes without any server or database setup.

---

## ✨ Features

| Feature | Description |
|---|---|
| ➕ Add Students | Register new students with their details |
| 🗑️ Delete Students | Remove student records instantly |
| 📋 Attendance Tracking | Mark and monitor attendance status per student |
| 📊 Marks Entry | Input subject-wise marks for each student |
| 🏅 Grade Calculation | Automatically compute grades based on marks |
| 💾 Persistent Storage | Data saved in browser via LocalStorage |

## 🚀 Getting Started

No installation or server needed. Just follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/student-management-system.git
   ```

2. **Navigate into the folder**
   ```bash
   cd student-management-system
   ```

3. **Open in browser**
   ```bash
   # Simply open index.html in any modern browser
   open index.html
   ```

> ✅ Works on all modern browsers — Chrome, Firefox, Edge, Brave.
